# Styles

## Theme tokens
- Background: `#0b1221`
- Panel: `#111827`
- Accent: `#38bdf8` (primary), `#22d3ee` (secondary)
- Text: `#e2e8f0`, Muted: `#94a3b8`
- Border: rgba(148, 163, 184, 0.18)
- Radius: 14px

## Typography
- Font: Inter, system fallback
- Headings: bold, compact; body size 14–16px; small helper text 13–14px muted color.

## Components
- **Panels**: dark glass background, border, subtle shadow, 14px radius.
- **Buttons**: gradient accent primary; secondary with transparent background and border.
- **Lists**: border + subtle background; used for activities and recent workshops.
- **Tags**: severity chips (critical/high/medium) with outline style.
- **Progress bar**: rounded, dual-gradient bar.

## Layout
- CSS grid utilities `.grid.two` and `.grid.three` for responsive cards.
- Sidebar width 240px; collapses under 720px.

## Accessibility
- High contrast accent color, focusable inputs, large click targets, minimal animations.
